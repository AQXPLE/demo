# MedConnect

Pakistan's healthcare directory — find doctors, book appointments, and explore health resources.

## Run locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy on Vercel

Push to GitHub and import the repo in [Vercel](https://vercel.com). The included `vercel.json` handles client-side routing.
